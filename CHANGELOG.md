# Changelog

## 0.1.0 - 2026-06-06

- Added a Unified settings modal with separate Chat API, Image API, and Video API tabs.
- Added Clear history actions for chats, generated images, and generated videos.
- Improved video generation feedback with staged status messages.
- Updated the video duration selector to support 1-15 seconds.
- Added the Image Creation workspace and local image history.
- Updated bilingual English and Chinese README documentation.
